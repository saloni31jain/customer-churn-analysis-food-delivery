# Customer Churn Analysis & Retention Strategy  
## Food Delivery Platform  

A behavioral analysis of customer churn using transactional data to identify key drivers of retention.

---

## 📌 Overview

This project analyzes customer behavior to understand why users stop ordering on a food delivery platform.

Using data from **4,339 customers**, the analysis identifies key churn patterns and provides business recommendations to improve retention.

---

## 📊 Key Results

- **33.4% churn rate**
- **65.3% of users placed fewer than 3 orders**
- Churn drops from **56.7% (low-frequency)** to **3.2% (high-frequency)**
- Low-value users churn significantly more than high-value users
- **14.9% users identified as at-risk**

---

## 🧠 Key Insight

Churn is primarily driven by:

- Low early engagement  
- Lack of repeat ordering behavior  
- Low perceived customer value
